# Verano Científico OAN-SPM 2026

## Simulación y monitoreo de componentes de un telescopio usando MQTT

Repositorio de trabajo para el proyecto del XXXV Verano Científico del Observatorio Astronómico Nacional en San Pedro Mártir (OAN-SPM).

### Objetivo

Implementar un entorno simplificado de instrumentación astronómica en el que distintos componentes simulados intercambien información mediante MQTT y visualizar su estado en tiempo real mediante dashboards desarrollados en Node-RED.

### Herramientas

- Python
- MQTT
- Node-RED
- Git y GitHub

### Estructura del proyecto

La estructura de directorios podrá modificarse durante el desarrollo del proyecto.

```
simulators/
dashboard/
docs/
examples/
```

### Actividades iniciales

- Configuración del entorno de desarrollo.
- Instalación de Python.
- Instalación de Git.
- Configuración de GitHub.
- Familiarización con el repositorio.
